Fixed Width Container

In the event that you wish to have a site with a more fixed layout, you can use the .container class. Below 768px, the container will fluidly remain at 100% width. The breakpoints above 768px are 992px and 1200px

Column Sizes

There are four different column sizes. Large, Medium, Small, and Extra Small. Barebones Grid is based off of a standard 12 column layout. To use the columns, simply give the class .{type}-* to the object. In the case of a medium sized column, if you wish the object to have half the container size, give the class .md-6.

Large Columns

Large columns are for screen sizes of 1200px and up. Anything under that size, without specifying a smaller column size, will cause the columns to stack. Use the class .lg-{num} where {num} is a number from 1 to 12.

Medium Columns

Medium columns are for screen sizes from 992px up to 1199px. Again, below that size and the columns will stack. Above that size, however, the columns will remain the same proportion with relation to the screen size. Use the class .md-{num} where {num} is a number from 1 to 12.
